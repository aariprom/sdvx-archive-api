### local (host execution)
```
$ npm run build
$ npm run prisma:generate
$ npm run prisma:migrate:local
$ npm run net:up
$ npm run db:up:local
$ npm run start:local
```

### local (container execution)
```
$ npm run build
$ npm run prisma:generate
$ npm run prisma:migrate:local
$ npm run net:up
$ npm run db:up:local
$ npm run dev:local
```

### dev (container execution)
```
$ npm run build
$ npm run prisma:generate
$ npm run prisma:migrate:dev
$ npm run net:up
$ npm run db:up:dev
$ npm run dev:dev
```
